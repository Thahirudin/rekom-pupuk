## Lakukan Instalasi di Terminal
- `composer install`

- `npm run dev`

## Aktifkan App Key
`php artisan key:generate`

## Migration Database
- Aktifkan Xampp MySQL atau dabase lainnya dan jalankan perintah di Terminal
`php artisan migrate`

## Jalankan Web
- Jalankan di Terminal `php artisan serve` dan `npm run dev`
- Web sudah bisa dibuka di chrome http://127.0.0.1:8000/
